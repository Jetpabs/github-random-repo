# Github Random Repo
A simple Github repository finder that returns a random repository based on the programming language input by the user.

**Link to roadmap.sh project:** https://roadmap.sh/projects/github-random-repo

## How It's Made
**Tech used:** HTML, CSS, JavaScript, Node.js, Express

HTML and CSS are used for the frontend design, JavaScript updates the DOM and handles API requests, and Node.js with Express manages server-side routing and retrieves data from the GitHub API.

## Lessons Learned
This project helped me practice implementing REST APIs with Node.js and Express, managing asynchronous API calls in JavaScript, and structuring a simple full-stack application.
